# 建構本地開發環境(Setting up a Local Development Environment)

![](https://github.com/dcpuse003/wordpress.github.io/blob/main/image/%E6%B7%B1%E8%97%8D01.jpg?raw=true)

#### 為 WordPress 設定本機開發環境，可以讓你在將網站部署到線上伺服器之前，先在自己的電腦上建立和測試網站。以下是關於如何為 WordPress 設定本機開發環境的一般指南。

1. #### 安裝本機伺服器環境：要在本機上執行 WordPress，您需要一個伺服器環境，例如 XAMPP（Windows、Mac、Linux）、MAMP（Mac）或 WAMP（Windows）。請根據您的作業系統選擇合適的軟體，並從其各自的網站下載。

2. #### 安裝本機伺服器軟體：按照伺服器軟體提供的說明將其安裝到您的電腦上。軟體將設定一個包含 Apache（Web 伺服器）、MySQL（資料庫）和 PHP（程式語言）的本機伺服器環境。

3. #### 啟動本機伺服器：安裝完成後，啟動本機伺服器軟體。它通常會有一個控制面板或介面，您可以在其中啟動必要的伺服器服務（Apache 和 MySQL）。

4. #### 建立資料庫：存取本機伺服器軟體的控制面板，並導覽至 MySQL 或資料庫部分。為您的 WordPress 安裝建立一個新資料庫。記下資料庫名稱、使用者名稱、密碼和主機（通常為 localhost）。

5. #### 下載 WordPress：造訪 WordPress 官方網站 (wordpress.org) 並下載最新版本的 WordPress。將下載的 zip 檔案解壓縮到您電腦上的一個資料夾。

6. #### 行動 WordPress 檔案：找到您安裝本機伺服器軟體的資料夾（例如，XAMPP 或 MAMP 的 htdocs 資料夾）。將解壓縮後的 WordPress 檔案移到此資料夾。

7. #### 設定 WordPress：將 WordPress 資料夾中的「wp-config-sample.php」檔案重新命名為「wp-config.php」。使用文字編輯器開啟“wp-config.php”文件，並輸入您先前記錄的資料庫詳細資訊（資料庫名稱、使用者名稱、密碼和主機）。

8. #### 造訪本機 WordPress 網站：開啟 Web 瀏覽器，並在 URL 中輸入「localhost」或「127.0.0.1」。如果您將 WordPress 資料放在子資料夾中，請在 URL 中包含資料夾名稱（例如，localhost/mywebsite）。此時應該會顯示 WordPress 安裝精靈。

9. #### 完成 WordPress 安裝：選擇您偏好的語言，並提供網站標題、使用者名稱、密碼和電子郵件地址以建立初始管理員帳戶。點擊“安裝 WordPress”按鈕繼續。

10. #### 存取本機 WordPress 控制面板：安裝完成後，您可以透過存取您在步驟 8 中輸入的 URL，並在其後加上「/wp-admin」（例如，localhost/wp-admin）來存取 WordPress 控制面板。使用您在安裝過程中設定的管理員憑證登入。

#### 恭喜！您已成功建置 WordPress 本機開發環境。現在，您可以開始在本地建立網站、安裝主題和外掛程式、測試功能並進行更改，而不會影響您的線上網站。請記得定期備份本地網站，準備就緒後，即可將網站部署到線上伺服器，供全球使用者存取。
