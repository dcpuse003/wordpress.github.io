安裝 WordPress

安裝 WordPress 的過程相對簡單。以下是一份關於如何在網頁主機伺服器上安裝 WordPress 的一般指南：

  1. 選擇網頁主機供應商(Choose a Web Hosting Provider)：挑選一家在價格、效能和功能方面符合您需求的網頁主機供應商。提供簡易 WordPress 安裝功能的熱門主機供應商包括 Bluehost、SiteGround 和 DreamHost。還有一些免費託管(會有一些限制，但拿來練手還不錯，有些還提供免費子網域，就是寄託在別人家的域名地址)，如InfinityFree,WordPress.com 等。

  2. 註冊網域名稱(Register a Domain)：如果您還沒有網域名稱，就需要註冊一個。許多網頁主機供應商都提供網域名稱註冊服務，您也可以透過 Namecheap 或 GoDaddy 等網域名稱註冊商單獨註冊網域名稱。說明域名，域名就像一個家的主要聯絡地址，需註冊公開讓所有人查找並連絡的上。

  3. 設定主機帳戶(Set Up Hosting Account)：向您選擇的供應商註冊主機帳戶。依照其指示設定帳戶、提供付款資訊，並完成必要的設定。

  4. 存取控制面板(Access Control Panel)：主機帳戶設定完成後，您通常會收到登入憑證以存取主機控制面板。最常見的控制面板是 cPanel，但不同主機供應商可能有其自訂的控制面板。

  5. 建立資料庫(Create a Database)：在控制面板中，尋找名為「資料庫」或「MySQL 資料庫」的區塊。建立一個新資料庫，並記下資料庫名稱、使用者名稱和密碼。這些資訊在 WordPress 安裝過程中將會用到。

     

  1. 下載 WordPress：造訪 WordPress 官方網站 (wordpress.org) 並下載最新版本的 WordPress。下載的檔案是壓縮套件 (.zip)，您可以將其儲存到您的電腦上。

  1. 上傳 WordPress 檔案：使用 FileZilla 等 FTP 用戶端連接到您的主機伺服器，並導覽至您網域的根目錄（通常為 public_html 或htdocs 或 www）。從下載的 zip 檔案中提取 WordPress 文件，並將其上傳到根目錄。

  2. 安裝 WordPress：開啟網頁瀏覽器並造訪您的網域（例如 yourdomain.com）。 WordPress 安裝精靈將會出現。選擇您喜歡的語言，然後點擊“開始(Let's Go)”按鈕。

  3. 設定資料庫設定：輸入您先前建立的資料庫詳細信息，包括資料庫名稱、使用者名稱、密碼和資料庫主機（通常為「localhost」）。除非有特殊原因需要更改，否則請將表前綴保留為預設值「wp_」。

  4. 執行安裝：點選「提交(Submit)」按鈕，WordPress 將檢查資料庫連線和設定。如果一切正常，您將看到成功訊息。點擊“執行安裝(Run the installation)”按鈕。

  5. 設定網站資訊：提供網站標題、使用者名稱、密碼和初始管理員帳戶的電子郵件地址。選擇一個強密碼來保護您的 WordPress 安裝至關重要。

  6. 完成安裝：點選「安裝 WordPress(Install WordPress)」按鈕，WordPress 將完成安裝。完成後，您將看到一條確認訊息，其中包含您的登入 URL。

  7. 存取 WordPress 控制面板：造訪登入 URL（通常為 yourdomain.com/wp-admin），輸入您的使用者名稱和密碼。您將被重定向到 WordPress 控制面板，在那裡您可以開始自訂網站、添加內容以及安裝主題和外掛程式。完成！您已成功在虛擬主機伺服器上安裝了 WordPress。現在，您可以開始使用WordPress 平台建立和管理您的網站。

    除了前面描述的手動安裝過程外，許多虛擬主機提供者還提供了一種更簡單的方法，即透過其控制面板（例如 cPanel）借助 Softaculous 等自動安裝程式來安裝 WordPress。

    以下是在 cPanel 中使用 Softaculous 安裝 WordPress 的方法：

    1. 登入您的虛擬主機提供者提供的 cPanel 帳戶。通常，您可以透過造訪 yourdomain.com/cpanel 並輸入您的憑證來登入。

    2. 在 cPanel 介面中尋找「Softaculous」或「WordPress Installer」圖示。

      它通常位於“軟體”或“自動安裝程式”部分。點擊該圖示即可存取 Softaculous 自動安裝程式。

    1. 進入 Softaculous 後，您將看到可用軟體應用程式清單。尋找 WordPress，它通常會突出顯示。點擊 WordPress 圖示繼續。

    2. 接下來，點選「安裝(Install)」按鈕開始安裝程序。

    3. 填寫安裝詳細資料：

      • 選擇要安裝 WordPress 的網域(Domain)。如果您有多個域名，請選擇要使用的域名。

      • 如果您想將 WordPress 安裝在根目錄（例如 yourdomain.com），請將「目錄(Directory)」欄位留空。如果您想將其安裝在子目錄（例如 ourdomain.com/blog），請輸入所需的目錄名稱。

      • 輸入網站名稱和描述(name and description)。

      • 設定管理員使用者名稱(admin username)和密碼(password)。請確保選擇安全的密碼。

      • 輸入您的管理員信箱地址(email address)。

    4. 根據需要自訂(Customize)網站設定。 Softaculous 提供選擇網站標題、標語以及啟用/停用外掛程式和主題的選項。

    5. 根據需要選擇一個主題。 Softaculous 在安裝過程中提供一系列免費主題供您選擇。您也可以跳過此步驟，稍後在 WordPress 控制面板中選擇主題。

    6. 填寫完所需資訊後，請仔細檢查以確保準確無誤。

    7. 點選「安裝(Install)」或「完成(Finish)」按鈕開始安裝程序。 Softaculous 將使用您提供的資訊安裝 WordPress。

    8. 安裝完成後，Softaculous 將顯示一則成功訊息，其中包含 WordPress 控制台的登入 URL。

    9. 造訪提供的 URL（例如 yourdomain.com/wp-admin）即可造訪您的 WordPress 網站，並使用您在安裝過程中設定的管理員使用者名稱和密碼登入。

在 cPanel 中使用 Softaculous 和 WordPress 自動安裝程式可以簡化安裝過程，讓您無需手動上傳檔案或設定資料庫即可快速輕鬆地設定 WordPress。

請注意，特定步驟可能會因您的網站寄存服務提供者及其提供的 cPanel 版本而略有不同。
